Cambio Control
  • Para seleccionar un modelo de control diferente, vamos a utilizar la letra “S” de nuestro teclado.
  • Para activar o desactivar el candado de trabajo, vamos a utilizar la letra “E” de nuestro teclado. Este candado si se encuentra activado, nos prohíbe mover el vehículo.
Control Ackermann
  • El control mediante Ackermann restringe a solamente mover la dirección de las llantas delanteras y ambas se mueven en la misma dirección, simulando el funcionamiento de un automóvil.
  • Utilizando las flechas del teclado podemos hacer que el vehículo tenga movimiento.
  • Utilizando las flechas de derecha o izquierda, (← , →), modificamos la dirección en las que las llantas delanteras se encuentran. Con una capacidad máxima de 70°
  • Utilizando las flechas de Adelante y Atrás, (↑ , ↓), podemos hacer que el vehículo se mueva en la dirección en la que estamos seleccionando.
Control Omnidireccional
  • El control mediante un sistema Omnidireccional nos permite mover la dirección de cada una de las llantas del vehículo. Dentro de nuestro sistema, solamente se puede mover las llantas por pares, las delanteras y traseras. Esto para restringir grados contrarios y que afecten el movimiento del vehículo.
  • Utilizando las flechas y el teclado podemos hacer que el vehículo tenga movimiento.
  • Para mover las llantas delanteras o frontales, debemos de seleccionar el modo frontal con la letra “F” de nuestro teclado.
  • Para mover las llantas traseras, debemos de seleccionar el modo trasero con la letra “T” de nuestro teclado.
  • Si nos encontramos en el modo frontal o trasero y queremos mover la dirección de las llantas, utilizamos las flechas derecha o izquierda, (← , →).
  • Si ya nos encontramos con la dirección que queremos, podemos mover el vehículo hacia adelante o atrás, utilizando las flechas (↑ , ↓) de nuestro teclado.
